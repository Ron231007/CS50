# 💰 Expense Tracker App
#### Video Demo: https://youtu.be/keUeBaY2Yuk

## 📌 Overview
This is a simple Python-based Expense Tracker application that helps users manage their daily finances. You can add expenses, update savings, set budgets, and view spending insights such as top expenses and budget status.

For simplicity, this project stores user data in a **CSV file**, allowing persistence across sessions. In a real-world or scaled-up version, this can be replaced with a proper database system.

---

## ✨ Features

1. ➕ Add Expense  
   - Record expenses with amount and category

2. 💾 Change Savings  
   - Update your current savings value

3. 👀 View Savings  
   - Check your current savings at any time

4. 📊 Check Budget Status  
   - Compare expenses against your budget

5. 💵 Change Budget  
   - Update your spending budget anytime

6. 🔝 See Top Expenses  
   - View the highest recorded expenses

7. 🚪 Exit  
   - Safely close the application

---

## 💾 Data Storage
- All expenses, savings, and budget data are stored in a **CSV file**
- This ensures data persistence even after the program is closed
- The CSV approach is used for simplicity and learning purposes

### 🔄 Future Improvement
If this project is scaled up, the CSV system can be replaced with:
- SQL Database (MySQL / PostgreSQL / SQLite)
- Cloud database solutions
- REST API backend for multi-user support
